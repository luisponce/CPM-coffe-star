Tabla de uso de la memoria
---------------------------
```DIR = DESCRIPTION```
  
 * 0xEE0 = 0 CONST
 * 0xEE1 = 1 CONST
 * 0xEE2 = 2 CONST
 * 0xEE3 = 3 CONST
 * 0xEE4 = Cantidad de empleados *(16 bits)*
 * 0xEE5 = Sueldo basico empleados *(32 bits)*
 * 0xEE6 = Reservado
 * 0xEE7 = Total pago a empleado de grupo *(32 bits)*
 * 0xEE8 = Reservado
 * 0xEE9 = Sueldo *(32 bits)*
 * 0xEEA = Reservado
 * 0xEEB = SMLMV *(32 bits)*
 * 0xEEC = Reservado
 * 0xEED = Numero salarios minimos *(16 bits)*
 * 0xEEE = Resevado
