Tabla de uso de la memoria
---------------------------
```DIR = DESCRIPTION```
  
 * EE0 = 0 CONST
 * EE1 = 1 CONST
 * EE2 = 2 CONST
 * EE3 = 3 CONST
 * EE4 = Cantidad de empleados *(16 bits)*
 * EE5 = Sueldo basico empleados *(32 bits)*
 * EE6 = Reservado
 * EE7 = Total pago a empleado de grupo *(32 bits)*
 * EE8 = Reservado
 * EE9 = Sueldo *(32 bits)*
 * EEA = Reservado
 * EEB = SMLMV *(32 bits)*
 * EEC = Reservado
 * EED = Numero salarios minimos *(16 bits)*
 * EEE = Resevado
